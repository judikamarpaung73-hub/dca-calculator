Bagaimana penyimpanan bekerja?

Input aktif tersimpan otomatis setiap Anda mengubah modal, periode, label, atau catatan. Untuk membuat jejak histori permanen, tekan tombol Simpan ke riwayat.

Catatan perhitungan

Profit CAGR menggunakan simulasi setoran bulanan dengan compounding bulanan. Dividen memakai pendekatan tahun pertama agar proyeksi tetap konservatif dan mudah dipahami.

Saran penggunaan

Buat snapshot berbeda untuk skenario gajian normal, bonus, atau target agresif. Nantinya Anda bisa membandingkan nominal dengan cepat dari panel histori.

Data histori tersimpan di perangkat Anda, bukan di server online. Jika cache browser dibersihkan, data lokal dapat ikut terhapus.
